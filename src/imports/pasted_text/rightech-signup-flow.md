Using the exact same premium SaaS design language as the login page, design a polished multi-step Sign Up flow for the RightTech Inventory App. The experience must feel high-end, structured, trustworthy, and modern, like serious enterprise SaaS software. Use only the Manrope font throughout the entire flow.

Overall design direction:
The sign up experience should feel premium, guided, calm, and intelligently structured. This is not a generic sign up page. It should feel like onboarding into a powerful inventory and operations platform built for Ghanaian businesses. Maintain the same visual identity, spacing system, border radius, typography hierarchy, and UI treatment used on the login page so the product feels fully consistent.

The UI should feel:
- Premium SaaS
- Clean and modern
- Spacious and elegant
- Enterprise-ready
- Friendly but professional
- Highly usable with strong visual hierarchy

Avoid:
- Cheap template feel
- Harsh gradients
- Overcrowded forms
- Playful or childish illustrations
- Inconsistent spacing
- Generic onboarding layouts

Sign up structure:
This is a multi-step progressive onboarding flow that opens when the user clicks “Sign Up.” The user should feel like they are being guided through a premium business account setup process.

Use a progress-based experience with:
- Step indicator or progress tracker
- Clear step titles
- Large elegant section headings
- Helpful supporting text
- Strong form structure
- “Back” and “Continue” actions where needed
- Smooth transitions between steps
- Consistent layout across all steps

Recommended layout:
Use the same design language as the login page, either in a centered premium card layout or a refined split-screen onboarding layout. The main form area should be clean and focused, while the side panel or top section can subtly reinforce the RightTech brand, inventory operations theme, or onboarding progress. The onboarding card should feel elevated, breathable, and very intentional.

Progress style:
Show that this is a guided setup process. Use a premium step progress UI, such as:
- top horizontal progress bar with step labels
- or step chips/cards with current step highlighted
- or a minimal “Step 01 of 05” system combined with a visual progress indicator

The progress must feel refined, not bulky.
